# GIF Screenshots for User Manual

Use a screen recording tool to capture GIF demos for each section.

**Recommended tools:** [ScreenToGif](https://www.screentogif.com/) (Windows) or [ShareX](https://getsharex.com/)

## Required GIFs

| Filename | Section | What to capture |
|----------|---------|-----------------|
| `getting-started.gif` | Getting Started | Launch app, setup wizard flow |
| `app-layout.gif` | App Layout | Pan across header, sidebar, content, right panels |
| `dashboard.gif` | Dashboard | Create project, view traffic lights, click into detail |
| `engineering-os.gif` | Engineering OS | Right-click create nodes, connect edges, context menu |
| `requirements.gif` | Requirements | Add/edit requirements, signal parameters, propagation |
| `console-panel.gif` | Console | Open console, see streaming logs, filter by level |
| `ai-assistant.gif` | AI Assistant | Open chat, send query, see streaming response |
| `ai-settings.gif` | AI Settings | Browse models, install, configure provider |

**Recording tips:**
- Resolution: 1280x720 or 1920x1080
- Frame rate: 10-15 FPS (keeps file size small)
- Duration: 5-15 seconds per GIF
- Use dark theme for consistency
